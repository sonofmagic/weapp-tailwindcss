简介

什么是 weapp-tailwindcss-webpack-plugin ？

这是一个让你在小程序中，使用 tailwindcss 大部分特性的一个 webpack 和 vite 插件集合。

它支持目前上几乎所有使用 webpack 和 vite 的主流多端小程序框架。

你可以轻而易举的在各个框架，或者原生开发中集成 tailwindcss。

现在就让我们赶紧开始使用吧！

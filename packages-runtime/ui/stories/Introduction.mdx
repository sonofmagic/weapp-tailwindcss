# @weapp-tailwindcss/ui

欢迎使用 **@weapp-tailwindcss/ui** 组件库！

## 项目概述

这是一个跨端的原子化 UI 组件库，专为微信小程序、Taro、uni-app 等多端框架设计。

### 核心特性

- 🎨 **原子化设计** - 基于 Tailwind CSS 的设计令牌系统
- 🔧 **Headless 架构** - 完全可定制的组件实现
- 🌍 **多端兼容** - 支持原生小程序、Taro、uni-app
- ⚡ **高性能** - JIT 编译，按需生成样式
- 🎯 **类型安全** - 完整的 TypeScript 支持

## 快速开始

### 安装

```bash
npm install @weapp-tailwindcss/ui
# 或
pnpm add @weapp-tailwindcss/ui
```

### 配置 Tailwind

在 `tailwind.config.ts` 中添加预设：

```typescript
import { weappTailwindcssUIPreset } from '@weapp-tailwindcss/ui/preset'

export default {
  content: ['./src/**/*.{ts,tsx,vue,html}'],
  presets: [weappTailwindcssUIPreset],
}
```

### 引入样式

```css
/* Tailwind CSS v4 项目 */
@import '@weapp-tailwindcss/ui/css';

/* Tailwind CSS v3 项目 */
@import '@weapp-tailwindcss/ui/tailwind3.css';
```

### 使用组件

```typescript
import { button } from '@weapp-tailwindcss/ui/variants'

// 生成按钮样式类名
const primaryButton = button({ tone: 'primary', appearance: 'solid' })
// => 'wt-button'
```

## 设计原则

### 原子化优先

所有组件都使用原子化的 CSS 类构建，确保：
- 样式可预测
- 易于覆盖
- 体积最小化

### Headless 组件架构

组件只提供逻辑和样式变体函数，UI 实现完全由开发者控制：
- 适配器系统支持多平台
- Hooks 提供可复用的状态逻辑
- 样式变体函数生成最优类名

### 跨端兼容

通过适配器模式实现：
- **Taro** - React 组件实现
- **uni-app** - Vue 组件实现
- **原生小程序** - 原生 WXML/JS 实现

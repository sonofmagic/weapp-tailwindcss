# uni-app-vue2-tailwind-hbuilder-template

`uni-app + Vue 2 + HBuilderX` 旧版模板，当前已过时，仅作为历史兼容示例保留。

## 当前状态

该模板基于较旧的 `uni-app webpack + postcss7` 体系，适合查看旧项目接入方式，不建议作为新项目起点。

> 该模板已过时，不再推荐继续投入新项目或新增功能。

更推荐直接使用：

- `uni-app-vite-vue3-tailwind-vscode-template`
- `uni-app-vue2-tailwind-vscode-template`
- `uni-app-vue3-tailwind-hbuilder-template`
- `uni-app-tailwindcss-v4`

## 技术栈

- `uni-app`
- `Vue 2`
- `Tailwind CSS postcss7 compat`
- `weapp-tailwindcss`
- `yarn`

## 使用方式

```bash
yarn
```

安装完成后，直接用 `HBuilderX` 导入项目并运行。

## 模板说明

- 该模板没有完整的命令行开发脚本，主要依赖 `HBuilderX`
- 如果新增了页面、组件或分包目录，需要同步更新 `tailwind.config.js` 的 `content`
- 这是兼容路径模板，不建议在新项目继续沿用旧技术栈

# uni-app-hbuilderx-tailwindcss-v4

`uni-app + HBuilderX + Tailwind CSS v4` 模板。

## 适用场景

- 使用 `HBuilderX` 管理 `uni-app` 项目
- 需要 `Tailwind CSS v4` 的模板起点
- 希望以最少脚本保留 `weapp-tailwindcss` 的补丁流程

## 技术栈

- `uni-app`
- `Tailwind CSS v4`
- `weapp-tailwindcss`
- `pnpm`

## 使用前提

- Node.js `20.19+`
- `pnpm`
- `HBuilderX`

## 快速开始

```bash
pnpm install
```

安装完成后，直接用 `HBuilderX` 导入项目并运行。

## 模板说明

- 安装依赖后会自动执行 `weapp-tw patch`
- 模板更偏 `HBuilderX` 工作流，命令行脚本较少
- 如果你偏好 VS Code + 命令行开发，优先考虑 `uni-app-tailwindcss-v4`

# @weapp-tailwindcss/babel

> English | [简体中文](./README.md)

This package provides the Babel primitives used internally by weapp-tailwindcss to parse, traverse, and validate JavaScript / TypeScript ASTs. It mainly supports internal transform and test flows rather than application-level usage.

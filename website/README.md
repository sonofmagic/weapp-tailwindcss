# Weapp-tailwindcss Website

docusaurus + tailwindcss

## 启动项目

```cmd
# 安装依赖
pnpm install

# 启动项目
pnpm dev
```

## 报错解决

Module not found: Error: Can't resolve '@weapp-tailwindcss/merge' in '/Users/zzy/Projects/weapp-tailwindcss/website/src/theme/ReactLiveScope'

```cmd
# 本地编译，cd 到根目录下

pnpm build

```

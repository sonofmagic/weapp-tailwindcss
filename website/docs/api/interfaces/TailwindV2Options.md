# TailwindV2Options

Tailwind CSS v2 补丁配置。

## 属性

### config?

> 可选 | **config**: `string`

Tailwind 配置文件路径。自动识别不够准确时可以显式传入。

***

### cwd?

> 可选 | **cwd**: `string`

解析 Tailwind 配置相对路径时使用的工作目录。

***

### postcssPlugin?

> 可选 | **postcssPlugin**: `string`

自定义 PostCSS 插件名称。未传入时使用默认名称。

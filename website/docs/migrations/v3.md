# Tailwind CSS 3 旧版迁移文档

当前文档站不再维护 Tailwind CSS 3 的接入、运行时与迁移内容。

如果项目必须继续使用 `tailwindcss@3`，请安装 `weapp-tailwindcss@4` 并查看 [v4 文档站](https://v4.tw.icebreaker.top/)。

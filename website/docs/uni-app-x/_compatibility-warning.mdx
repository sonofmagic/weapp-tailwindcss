:::warning 当前请使用 Tailwind CSS 3
uni-app x 的 Tailwind CSS 4 新链路仍有部分适配尚未完善。在这些问题解决前，请使用 `tailwindcss@3` 和 `weapp-tailwindcss@4`。

请参考 [weapp-tailwindcss v4 的 uni-app x 教程](https://v4.tw.icebreaker.top/docs/uni-app-x)。本页的 Tailwind CSS 4 内容仅供提前验证，暂不建议用于生产项目。
:::

:::info 5.2.0 起的运行环境
使用 `weapp-tailwindcss@5.2.0` 或更高版本进行验证时，需要 Node.js `>=22.12.0` 和 HBuilderX `>=5.11`。Node.js 22 从 `22.12.0` 开始默认支持从 CommonJS 加载 ESM；较旧的 HBuilderX 内置 Node 可能无法加载当前依赖。
:::

# 安装依赖

这一页只处理包安装。框架注册方式放到下一章。

## 安装当前版本

当前文档面向 `tailwindcss@4` 与最新版 `weapp-tailwindcss`。如果项目必须继续使用 `tailwindcss@3`，请安装 `weapp-tailwindcss@4` 并查看 [v4 文档站](https://v4.tw.icebreaker.top/)。

```bash npm2yarn
npm install -D tailwindcss weapp-tailwindcss
```

下一步：进入 [各框架注册方式](/docs/quick-start/frameworks/uni-app-vite)。

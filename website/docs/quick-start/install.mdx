# 安装依赖

这一页只处理包安装。框架注册方式放到下一章。

## 选择 Tailwind CSS 版本

如果你不确定选哪个版本，先用 `tailwindcss@3`。它对存量小程序项目更稳。

新项目想用 Tailwind CSS 4 的 CSS-first 写法，再安装最新版 `tailwindcss`。

## Tailwind CSS 3.x

```bash
pnpm add -D tailwindcss@3 weapp-tailwindcss
pnpm exec tailwindcss init
```

## Tailwind CSS 4.x

```bash
pnpm add -D tailwindcss weapp-tailwindcss
```

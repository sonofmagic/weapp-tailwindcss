`weapp-tailwindcss@5` 中，`weapp-tw patch` 已无需执行：

:::tip 删除旧的 postinstall
`weapp-tailwindcss@5` 已由 `WeappTailwindcss` 插件或生成器入口在构建运行时接管 Tailwind CSS，不需要再配置 `postinstall: "weapp-tw patch"`。
:::

```bash npm2yarn
npx weapp-tw patch
```

该命令现在只是兼容旧脚本的 no-op 提示，不再写入补丁记录、清理缓存或扫描 workspace。请把排障重点放到入口 CSS、`@source`、`cssEntries` 和构建插件是否生效上。

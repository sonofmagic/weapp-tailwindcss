# uni-app cli vue2 webpack

:::caution
这条 `uni-app cli vue2 webpack` 路线已经不推荐，仓库中的对应 classic demo 也已经移除。

请优先使用 [uni-app cli vue3 vite](/docs/quick-start/v4/uni-app-vite)。
:::

## 安装

```bash npm2yarn
npm install -D tailwindcss weapp-tailwindcss
```

## 配置

### 创建 `vue.config.js`

```js title="vue.config.js"
const { WeappTailwindcss } = require('weapp-tailwindcss/webpack')
const path = require('node:path')
/**
 * @type {import('@vue/cli-service').ProjectOptions}
 */
const config = {
  // some option...
  configureWebpack: (config) => {
    config.plugins.push(
      new WeappTailwindcss({
        rem2rpx: true,
        cssEntries: [
          // Tailwind CSS 入口文件绝对路径
          path.resolve(__dirname, './src/app.css'),
        ],
      })
    )
  }
  // other option...
}

module.exports = config
```

> tailwindcss@4 必须配置 `cssEntries` 并且使用绝对路径，否则 `tailwindcss` 生成的类名不会参与转译。

## PostCSS 配置

`weapp-tailwindcss@5` 默认由 Webpack 插件接管 Tailwind CSS 样式生成，不需要再在 `postcss.config.js` 里注册 `@tailwindcss/postcss`。如果这个存量项目已经有 PostCSS 配置，只保留 `postcss-import`、`autoprefixer`、`@dcloudio/vue-cli-plugin-uni/packages/postcss` 等非 Tailwind 插件即可。

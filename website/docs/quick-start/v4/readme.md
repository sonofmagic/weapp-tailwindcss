# Tailwind CSS 4 旧入口已迁移

`/docs/quick-start/v4` 不再作为快速开始入口，也不再表示 Tailwind CSS 4 是可选或实验性的额外接入方式。当前文档默认使用 Tailwind CSS 4 生成模式；CSS-first、`@source`、`@apply`、`@layer`、IntelliSense 与多端补充说明，已经迁移到 Tailwind 专题下。

请前往 [Tailwind CSS 4 默认模式参考](/docs/tailwindcss/v4-reference)。

如果你是第一次接入项目，请从 [安装依赖](/docs/quick-start/install) 开始，再进入对应框架注册页。

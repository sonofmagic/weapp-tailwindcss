# 开发参考手册已迁移

`/docs/quick-start/v4` 不再作为快速开始入口。Tailwind CSS 4 的 CSS-first、`@source`、`@apply`、`@layer`、IntelliSense 与多端补充说明，已经迁移到 Tailwind 专题下的开发参考手册。

请前往 [Tailwind CSS 4.x 开发参考](/docs/tailwindcss/v4-reference)。

如果你是第一次接入项目，请从 [安装依赖](/docs/quick-start/install) 开始，再进入对应框架注册页。
